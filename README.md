## 安装内容

- ✅ Python 3.11 + Alpha 虚拟环境
- ✅ PM2 进程管理器
- ✅ Google Chrome
- ✅ xbx-py11 库
- ✅ 中文字体（解决乱码）

## 环境路径

### Anaconda
```bash
# Python 解释器
$HOME/anaconda3/envs/Alpha/bin/python

# 激活环境
conda activate Alpha
```

### pyenv
```bash
# Python 解释器
$HOME/.pyenv/versions/3.11.0/envs/Alpha/bin/python

# 激活环境
source ~/.bashrc
pyenv activate Alpha
```

## 常用命令

```bash
# 查看应用状态
pm2 status

# 查看日志
pm2 logs

# 重启应用
pm2 restart all
```
